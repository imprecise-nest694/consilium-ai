## 🔍 Troubleshooting

### The app will not open

Try this:

- Right-click the file and choose Run as administrator
- Make sure the app was fully extracted if it came in a zip file
- Check that your antivirus did not block the file
- Download the file again if it looks incomplete

### The app opens but feels slow

Try this:

- Close other programs
- Use a smaller model
- Lower the context size
- Restart the app
- Make sure your system has enough free RAM

### The model does not load

Try this:

- Check that the model file is in the right folder
- Download the model again if the file was damaged
- Make sure you picked a model that fits 8GB RAM
- Reopen the app after the model download finishes

### Windows shows a security prompt

This can happen with downloaded apps. If you trust the source, choose the option to run the file.

### Text gets cut off

Try this:

- Start a new chat
- Use shorter prompts
- Reduce the amount of pasted text
- Check whether the app has a message length limit
